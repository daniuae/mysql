
# In this project, I built a Retail Sales Data Warehouse using Snowflake. First, I cleaned and transformed raw sales data using 
# Pandas by handling null values, duplicates, and formatting issues. Then I designed a Star Schema with Fact_Sales and multiple 
# Dimension tables such as Customer, Product, Date, and Store. After loading the data into Snowflake, I wrote SQL queries to 
# analyze sales performance, revenue trends, and customer behavior. Finally, I used Matplotlib and Seaborn to create visualizations 
# for monthly revenue, product performance, and regional sales analysis. 
# This project helped me gain hands-on experience in data warehousing, ETL development, SQL analytics, and data visualization.
